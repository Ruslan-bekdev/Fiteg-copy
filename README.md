# Fiteg Landing Page (High-Fidelity Clone)

A pixel-perfect recreation of a modern landing page, focused on complex layouts, smooth performance, and advanced styling techniques.

## [Live Demo](https://ruslan-bekdev.github.io/Fiteg-copy/)

## Key Challenges & Solutions
The main goal was to replicate a high-end design with heavy emphasis on animations and precise positioning without relying on massive UI libraries.

* **Advanced Layout Engineering:** Recreated complex grids and flexible containers using modern CSS and SASS, ensuring every element aligns perfectly across all screen sizes.
* **Custom Marquee System:** Implemented a high-performance "running text" (marquee) effect using vanilla CSS animations, avoiding the overhead of external JS plugins.
* **Styled Components Integration:** Experimented with CSS-in-JS patterns to manage dynamic styles and keep the component architecture clean and encapsulated.
* **Animation Polish:** Focused on smooth scroll transitions and hover effects to maintain the "premium" feel of the original design.

## Tech Stack
* **React** (Component Architecture)
* **Styled Components** (Dynamic Styling)
* **SASS** (Complex Layouts & Global Styles)
* **Vite** (Build Tool)

## Project Purpose
This project served as a deep dive into high-fidelity UI cloning, testing my ability to translate a sophisticated visual design into a functional, responsive, and performant web application.

## How to run locally
1. `git clone https://github.com/Ruslan-bekdev/fitegcopy.git`
2. `npm install`
3. `npm start`
